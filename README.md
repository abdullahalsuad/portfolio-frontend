- 🛣️ React Router v7
- 🧹 ESLint + Prettier
- 📱 Fully responsive
- 🚀 Production-optimized

---

## 🧰 Installation & Setup

```bash
git clone https://github.com/your-username/your-portfolio-frontend.git
cd your-portfolio-frontend
yarn
yarn run dev
```

Runs app at `http://localhost:5173`
